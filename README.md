# Concurrency Models

Proyecto Rust para la **Actividad 6 - Modelos de Concurrencia** de Programación Concurrente.

## Contenido

- [`docs/actividad-6-consigna.md`](docs/actividad-6-consigna.md): consigna formateada en Markdown.
- `src/main.rs`: punto de entrada del proyecto.

## Ejecución

```bash
cargo run
```

## Modelos tratados

- Fork-Join
- Async/Await
- Pasaje de mensajes

## Crates previstos

- `tokio`: ejercicios async/await.
- `rayon`: ejercicios Fork-Join con work-stealing.
